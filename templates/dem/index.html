GEOGRID

GeoGrid is a Web-based tool to query, form, comprehend and analyze Digital Elevation Models(DEM) in Real-time using Web Servers to simplify field work and diminish the sophistication involved in using similar office tools.

We can use these DEMs(Digital Elevation models) to get a qualitative and quantitative analysis and data for constructional challenges and to solve real world problems like Flood Control and Slope Analysis.
